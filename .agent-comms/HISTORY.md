## How to Use

After a feature/fix is **validated and merged**, add an entry below with:
- **What** changed (files, configs, behavior)
- **Why** it was needed (the problem it solves)
- **Lessons Learned** (gotchas, insights for future)
- **Commits** (for reference)

> **This is NOT a changelog for users.** This is persistent memory for AI agents to understand project evolution.
